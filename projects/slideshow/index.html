I have written down what I wear every day since October 5, 2020. My personal data collection practice evolved out of a graduate school project around the same theme, initially trying to assess how much black and gray I wear. It has since yielded the raw material for an ongoing set of process-oriented experiments. The current iteration investigates the following:

  1. Abstracting visual information to reveal larger patterns

  2. Inviting discovery through reorganization

  3. Building an interconnected system of visualizations

The data collection process included scanning every item that I wore.

Using the Color Palette From Image app, I pulled 5 representative colors from each scan.

These colors became an abstracted symbol for the item.

From the 3 items in an outfit:

jeans

sweater

coat

I could reimagine an impression of colors.
Much like turning an object in your hands, reorganizing this abstraction along different scales reveals slightly different facets of the data.
When I started to combine these daily symbols into larger buckets of time, I noticed to was hard to extract meaning. The data began to feel too noisy.
Aggregating smoothes the impression and creates a kind of derived dataset: a weekly cadence of colors.

Distributing these building blocks into buckets by hue, saturation, and lightness makes larger patterns visible.

I still update this dataset daily. It currently contains almost 67 weeks of entries.
